### 입력 

 <p>첫 줄엔 문자열의 높이 N과 가로 M이 주어진다. (2 ≤ N, M ≤ 1,000) 이다.</p>

<p>두 번째 줄부터 M개의 숫자 0또는 1이 예제 입력과 같이 N개의 줄에 걸쳐 입력된다.</p>

<p>1는 항상 두 개만 입력된다.</p>

### 출력
